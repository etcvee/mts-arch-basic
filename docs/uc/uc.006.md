<table>
    <thead>
        <th>Название</th>
        <th>Рассмотрение доклада</th>
    </thead>
    <tbody>
        <tr>
            <td>Описание</td>
            <td>Эксперт рассматривает доклад и оценивает насколько доклад соответствует тематике конференции, насколько он качественный и можно ли использовать его в программе конференции</td>
        </tr>
        <tr>
            <td>Участники</td>
            <td>Эксперт</td>
        </tr>
        <tr>
            <td>Триггер</td>
            <td>Пользователь находится в личном кабинете эксперта</td>
        </tr>
        <tr>
            <td>Предусловия</td>
            <td>
                1. Пользователь зарегистрирован на конференции в качестве эксперта<br/>
                2. Доклад подан на рассмотрение<br/>
                3. Доклад не рассматривается другим экспертом
                4. Доклад не утвержден<br/>
                5. Доклад не отклонен<br/>
            </td>
        </tr>
        <tr>
            <td>Постусловия</td>
            <td>Доклад перешел в статус "На рассмотрении"</td>
        </tr>
        <tr>
            <td>Успешный сценарий</td>
            <td>
                6. Система отображает список зарегистрированных докладов, ожидающие рассмотрения<br/>
                7. Пользователь выбирает доклад, который будет рассматривать<br/>
                8. Система назначает пользователя экспертом, рассматривающим текущий доклад<br/>
                9. Система переводит доклад в статус "На рассмотрении"
            </td>
        </tr>
        <tr>
            <td>Требования</td>
            <td>FR.009, FR.011</td>
        </tr>
    </tbody>
</table>
